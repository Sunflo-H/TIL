# ✅ Next.js란?

- <b>웹 어플리케이션을 만들기 위한 React Framework</b>
- React로 웹 앱을 개발하는데 리액트의 한계를 보완하는 다양한 기능들이 존재

### 🤔 왜 사용하나요?

앞서 말한대로 React의 한계를 보완한다.
CSR의 단점을 극복하고, SSR(및 SSG 등 하이브리드 렌더링)을 사용합니다.

- CSR 단점1 : 초기 로딩속도 느림
- CSR 단점2 : SEO 안좋음
